# Xbox Controller for Java UI

## Overview

This is a graphical JavaFX implementation that utilizes the XboxControllerForJava library. It displays all major events from the Xbox controller in a user-friendly interface.

## Screenshot

![Application Screenshot](https://github.com/NeuralCortex/XboxControllerForJavaUI/blob/main/images/app.gif)

## Features

| Feature             | Description                                                      |
|---------------------|------------------------------------------------------------------|
| Connectivity        | Events for up to 2 players are displayed (easily extendable to 4) |
| Supported Functions | Triggers, buttons, analog sticks, and vibration                  |

## Minimum Requirements

- Windows 11 Home
- Java Runtime Environment (JRE) or Java Development Kit (JDK) version 25

## Usage Instructions

- Simply connect your Xbox controller via USB cable or Bluetooth. The events will appear in the UI.
- A precompiled, ready-to-use JAR file is included.

## Technologies Used

- **IDE**: NetBeans IDE 28 [NetBeans](https://netbeans.apache.org/front/main/index.html)
- **Java SDK**: Java 25 [Java 25](https://www.oracle.com/java/technologies/downloads/#jdk25-windows)
- **JavaFX SDK**: JavaFX [JavaFX SDK 26](https://openjfx.io/)
- **Scene Builder for UI**: Gluon Scene Builder [Scene Builder 26](https://gluonhq.com/products/scene-builder/)
